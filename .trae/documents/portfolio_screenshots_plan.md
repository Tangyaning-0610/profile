# 成果截图展示模块实现计划

## 一、推荐形式分析

### 方案：首页精选成果画廊（推荐）
- **布局**：网格布局 + 悬浮放大效果
- **特点**：视觉冲击力强，展示跨项目亮点成果
- **位置**：放在"工作经历"模块之后

## 二、实现步骤

### 步骤1：创建成果画廊组件（Gallery.astro）
- 创建网格布局展示截图
- 添加悬浮放大效果
- 支持点击查看大图（Lightbox）

### 步骤2：更新首页（index.astro）
- 引入 Gallery 组件
- 放在 WorksList 之后

### 步骤3：准备示例截图
- 在 public/assets 目录下创建示例截图或占位图

## 三、文件修改清单

| 文件 | 修改类型 | 说明 |
|------|----------|------|
| src/components/Gallery.astro | 新建 | 成果画廊组件 |
| src/pages/index.astro | 修改 | 引入 Gallery 组件 |

## 四、技术要点

1. **图片懒加载**：使用 native lazyload
2. **响应式布局**：移动端自动调整列数
3. **悬浮效果**：hover 时放大并显示阴影
4. **Lightbox**：点击图片弹出大图查看

## 五、风险处理

- 如果用户暂无截图，默认显示占位图
- 确保图片路径正确，统一存放在 public/assets 目录
